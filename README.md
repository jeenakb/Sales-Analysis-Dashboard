                                                 Sales & Profitability Analysis Dashboard
                                                          
Overview

This Power BI project provides a comprehensive analysis of sales performance and financial health for a multi-category retail environment. The dashboard is designed to help stakeholders identify high-growth products, monitor profit margins, and evaluate the effectiveness of discount strategies.

Key Insights from the Dashboard

Revenue concentration is driven by specific top-performing brands, indicating a healthy but concentrated market position. While high discounts drive volume in certain categories, they show a diminishing return on net profit in the specific segments analyzed. Despite lower sales volume, the lead product categories maintain the highest profit-to-sales ratio. The dashboard utilizes a high-contrast palette to ensure KPIs like Total Sales and Net Profit are immediately distinguishable for executive review.

Business Questions Addressed

•	Product Performance: Which specific brands or products are the primary drivers of revenue? 
•	Discount Strategy: Are current discount levels effectively boosting sales without eroding profit margins? 
•	Market Share: How is the quantity distributed across different product categories? 
•	Operational Health: What is the total profit standing after accounting for all applied discounts and costs?

Technical Process

•	Data Preparation: Processed raw sales records using Power Query to handle missing values and ensure data consistency. 
•	Data Modeling: Established a streamlined data model to link product categories, brands, and financial metrics. 
•	Visualizations: Used KPI Cards for instant visibility, Donut Charts for discount breakdowns, and Data Tables for row-level analysis. 
•	Design: Migrated from a monochromatic layout to a high-contrast, semantic color palette to improve data storytelling and user experience.

How to Use

1.	Clone this repository: git clone https://github.com/jeenakb/Sales-Analysis-Dashboard.git
2.	Open the .pbip file in Power BI Desktop.
3.	If data links are broken, update the Data Source Settings to point to the data folder included in this repository.
4.	Use the slicers at the top to filter the analysis by Brand, Product, or Category.
